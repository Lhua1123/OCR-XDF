# 课程信息提取工具 —— 需求文档

## 1. 项目概述

### 1.1 项目背景
在日常学习/工作中，用户需要从课程截图（如网课平台、直播课堂等）中提取课程信息（课程名称、时间、地点等），并生成格式化通知文本。当前流程依赖手动操作，效率低下。本工具旨在实现 **截图 → AI视觉识别 → 通知生成** 全流程自动化。

### 1.2 项目目标
开发一个带 GUI 界面的桌面工具，能够：
- 通过粘贴图片获取课程表截图
- 使用视觉大模型（VLM）直接从图片中提取课程信息
- 自动按日期排序课程
- 生成可直接发送的课程通知文本
- 支持自定义通知模板

### 1.3 目标用户
需要频繁记录/分享课程安排的学生、教务人员、助教等。

## 2. 功能需求

### 2.1 图片输入模块
| 编号 | 功能 | 描述 |
|------|------|------|
| F-01 | 粘贴图片 | 支持 Ctrl+V 从剪贴板粘贴图片 |
| F-02 | 多图支持 | 最多同时处理5张图片 |
| F-03 | 图片预览 | 在界面上显示已粘贴图片的缩略图 |
| F-04 | 清空图片 | 一键清空所有已粘贴的图片 |

### 2.2 AI 识别模块
| 编号 | 功能 | 描述 |
|------|------|------|
| F-05 | 视觉识别 | 通过视觉大模型（VLM）直接从图片提取课程信息 |
| F-06 | 多图合并 | 多张图片的课程自动合并去重 |
| F-07 | 自动排序 | 课程按日期自动排序 |
| F-08 | 自定义模板 | 支持自定义通知模板，AI严格按照模板格式输出 |

### 2.3 通知生成模块
| 编号 | 功能 | 描述 |
|------|------|------|
| F-09 | 文本生成 | 根据识别结果生成格式化通知文本 |
| F-10 | 模板管理 | 支持自定义通知模板，保存后自动同步到提示词 |
| F-11 | 预览与编辑 | 在界面上预览生成的通知文本，并支持直接编辑 |
| F-12 | 复制到剪贴板 | 一键复制通知文本到系统剪贴板 |
| F-13 | 保存到文件 | 将通知文本保存到本地文件（.txt） |

### 2.4 历史记录
| 编号 | 功能 | 描述 |
|------|------|------|
| F-14 | 自动保存 | 自动保存处理记录 |
| F-15 | 历史查询 | 查看历史处理记录（截图、通知文本） |

### 2.5 系统设置
| 编号 | 功能 | 描述 |
|------|------|------|
| F-16 | API配置 | 配置 API Key、API Base URL、模型名称 |
| F-17 | 连接测试 | 测试 API 连接是否正常 |
| F-18 | 通知模板 | 自定义通知模板格式 |
| F-19 | 主题切换 | 支持亮色/暗色主题切换 |

## 3. 非功能需求

### 3.1 性能
- 图片粘贴响应时间 < 1 秒
- AI 识别完成时间 < 30 秒（视 API 响应速度和图片数量）
- 多图处理支持并行识别

### 3.2 可用性
- GUI 界面简洁直观，操作流程清晰
- 每一步都提供明确的进度反馈
- 支持 Ctrl+V 快捷键粘贴图片

### 3.3 可靠性
- AI 调用失败时，给出明确错误提示
- 自动保存当前工作状态
- 支持日志记录，方便排查问题

### 3.4 兼容性
- 操作系统：Windows 10/11

### 3.5 安全性
- API Key 存储在本地配置文件，不硬编码
- 所有数据处理在本地完成

## 5. 用户界面设计

### 5.1 主界面布局

```
┌─────────────────────────────────────────────────────┐
│  菜单栏：工具（设置、历史记录）| 帮助                  │
├─────────────────────────────────────────────────────┤
│  [粘贴图片] [清空]              图片：0/5             │
├──────────────────────┬──────────────────────────────┤
│                      │  步骤提示                     │
│                      │  [开始识别]                   │
│   图片预览区域        │                              │
│   (支持多图显示)      │  通知文本预览/编辑框           │
│                      │                              │
│                      │  [复制到剪贴板] [保存到文件]    │
│                      │                              │
│                      │  ┌──────────────────────┐    │
│                      │  │ 提取信息预览          │    │
│                      │  └──────────────────────┘    │
├──────────────────────┴──────────────────────────────┤
│  状态栏：当前状态 / 提示信息                          │
└─────────────────────────────────────────────────────┘
```

### 5.2 交互流程

```
用户按 Ctrl+V 粘贴图片
    → 图片显示在预览区
    → (可选) 继续粘贴更多图片
    → 点击 [开始识别]
    → AI 自动提取课程信息
    → 课程按日期排序
    → 通知文本显示在编辑区
    → 用户 [复制] 或 [保存]
```

## 6. 数据格式

### 6.1 通知文本示例

```
📚 课程通知
━━━━━━━━━━━━━━━━━━
📅 日期：5月1日 周四 | 📖 科目：语文 | ⏰ 时间：14:00-16:00（下午） | 📍 位置：VIP203
📅 日期：5月2日 周五 | 📖 科目：数学 | ⏰ 时间：14:00-16:00（下午） | 📍 位置：VIP201
📅 日期：5月3日 周六 | 📖 科目：数学 | ⏰ 时间：10:00-12:00（上午） | 📍 位置：VIP203
━━━━━━━━━━━━━━━━━━
此条信息不用回复，如有特殊情况提前沟通
```

### 6.2 自定义模板说明

- 分隔线 `━━━━━━━━━━━━━━━━━━` 之间的课程行会自动按日期排序
- 分隔线之外的内容（标题、页脚）会完整保留
- 用户可在设置中自定义整个模板格式
